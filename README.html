Introduction

CredentialHub is a robust application developed to help users manage and organize their password resets and credentials across various online services. Whether you're handling multiple email accounts, social networks, cloud storage, or e-commerce platforms, CredentialHub provides a centralized and secure way to keep track of your reset links and credentials.

Features

  • Categorized Management: Organize services into categories like Email Services, Social Networks, Cloud Storage, and more.
  • Preloaded Reset Links: Access quick reset links for popular services.
  • Secure Credential Storage: Save and manage usernames, passwords, and associated dates securely.
  • User-Friendly GUI: Intuitive interface built with Tkinter for easy navigation and management.

Installation

1. Clone the Repository

git clone https://github.com/yourusername/CredentialHub.git
cd CredentialHub

2. Install Required Packages

pip install -r requirements.txt

Usage

1. Run the Application

python "Security Check - Application (GVTech Hub).py"

2. Using CredentialHub

  • Categories: Select a category to view associated services.
  • Sites: Choose a site to view or edit its reset URL and credentials.
  • Save Credentials: Enter your username, password, and date, then click "Save Credentials" to store them securely.

Requirements

Ensure you have Python installed. Then, install the necessary packages:

pip install tkinter

Note: tkinter usually comes pre-installed with Python. If not, install it using your package manager.

Example Scenario

Customer Situation:

A small business owner manages multiple online accounts for email, cloud storage, social media, and e-commerce platforms. Keeping track of password reset links and credentials becomes cumbersome and insecure over time.

Using CredentialHub:

  • Organize: All services are organized into relevant categories.
  • Access: Quick reset links are available for each service when needed.
  • Store: Usernames, passwords, and dates are stored and updated securely.
  • Manage: Easily retrieve and manage credentials, ensuring better security and efficiency.
